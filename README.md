### System Design
* ###### Use Design Pettern
* ###### Architecture Pattern
    * ###### With MVVM
* ###### Use Clean Architecture
    * ###### Ui -> Framework
    * ###### Presenter -> Interface Adapter
    * ###### UseCase -> Application Business Rule
    * ###### Entities -> Enterprise Business Rule
    * ###### Repository -> Interface Adapter
    * ###### DataSource -> Framework
